# Sketch

MakeTime 내용을 실생활에 적용하기 위한 개인 Small Project **Sketch**

## 컨셉

**즐거움, 만족감, 개인을 활기차게하는 습관 정착시키기**  

**SIMPLE**

**반응형**

## 기능 
- Top 화면
	- 로그인(TODO)

- 왼쪽 Page
	- 날짜 표시

	- 진행여부 체크 box

	- sorting
	- 일괄편집
